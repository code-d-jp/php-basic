## ex_var05.php

変数の平均値を出力するプログラムを作成します。
次の結果のとおり出力するプログラムを作成してください。

### 結果


```
AVERAGE:30
```

### プログラム

```
<?php
$age1 = 22;
$age2 = 28;
$age3 = 40;
$average = ($____ + $____ + $____) / _;
echo "AVERAGE:" . $average;
```

<details>
<summary>解答</summary>

```
<?php

$age2 = 28;
$age3 = 40;

echo "AVERAGE:" . $average;
```
